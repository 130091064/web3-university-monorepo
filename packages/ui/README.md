# @lillianfish/ui

UI component library based on Radix UI, React, TypeScript and Tailwind CSS.

## Installation

```bash
pnpm add @lillianfish/ui
```

## Usage

```tsx
import { Button } from '@lillianfish/ui';
import '@lillianfish/ui/styles.css';

function App() {
  return <Button variant="primary">Click me</Button>;
}
```

## Components

- Button

## Development

```bash
# Build the library
pnpm build

# Watch mode
pnpm dev
```
